# AMICIZIA – Website 🍕

Moderne, animierte Website für **AMICIZIA** – ein Familien-Foodspot in Saarlouis mit frischer Pizza, Pasta und mehr.

## Tech Stack

- **Next.js 16** – App Router, TypeScript
- **Tailwind CSS v4** – Utility-first styling
- **Framer Motion** – Smooth scroll-triggered animations
- **Geist Font** – Clean, modern typography

## Design

- **Farben:** Bordeaux Rot (`#6B1024`) + Weiß
- **Style:** Clean, sexy, minimalistisch
- **Animations:** Scroll-reveal, parallax shapes, marquee, hover effects

## Sections

1. **Hero** – Fullscreen mit animiertem Hintergrund und CTAs
2. **Marquee** – Endlos-Laufband mit Keywords
3. **Über uns** – Familiengeschichte mit Statistiken
4. **Speisekarte** – Filterbares Menü-Grid (Pizza, Pasta, Vorspeisen, Extras)
5. **Lieferung & Abholung** – Feature-Cards mit Lieferando-Link
6. **Kontakt** – Adresse, Öffnungszeiten, Google Maps
7. **Footer** – Quick Links & Social Media

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
